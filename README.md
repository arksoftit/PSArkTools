# PSarktools 0.1.1

Conjunto de scripts en PowerShell para soporte técnico rápido en entornos Windows.

## Características
- Diagnóstico rápido del sistema, red y discos.
- Escaneo de puertos y conexiones activas.
- Reparaciones comunes (DNS, red, servicios).
- Gestión de usuarios y auditoría de administradores.
- Totalmente portable: funciona desde USB.

## Requisitos
- Windows 10/11 o Windows Server 2016+
- PowerShell 5.1 o superior
- Ejecución con permisos de administrador (recomendado)

# PSarktools Ver. 0.1.2

Conjunto de scripts en PowerShell para soporte técnico rápido en entornos Windows.

## Características
- Diagnóstico rápido del sistema, red y discos.
- Escaneo de puertos y conexiones activas con nombres de procesos.
- Reparaciones comunes (DNS, red, servicios críticos).
- Gestión de usuarios y auditoría del grupo de administradores.
- **Configuración regional internacional** (separadores, formato de fecha, moneda).
- Totalmente portable: funciona desde USB sin instalación.

## Requisitos
- Windows 10/11 o Windows Server 2016+
- PowerShell 5.1 o superior
- Ejecución con permisos de administrador (recomendado para reparaciones y auditoría completa)

## Uso
```powershell
.\SoporteTool.ps1          # Menú interactivo
.\SoporteTool.ps1 -Quick   # Modo rápido en consola
.\SoporteTool.ps1 -Report  # Genera reporte en ./Reportes/
